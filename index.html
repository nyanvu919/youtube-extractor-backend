Cách lấy YouTube API Key miễn phí (Chi tiết từng bước):

Bước 1: Truy cập vào Google Cloud Console và đăng nhập bằng tài khoản Gmail của bạn.

Bước 2: Tạo dự án mới (Project):
- Nhấn vào ô chọn dự án ở góc trên cùng bên trái (cạnh chữ Google Cloud).
- Chọn "New Project" (Dự án mới), đặt tên bất kỳ (ví dụ: YoutubeTool) và nhấn "Create".

Bước 3: Kích hoạt YouTube API:
- Tại thanh tìm kiếm trên cùng, gõ chữ "YouTube Data API v3".
- Chọn kết quả hiện ra và nhấn nút xanh "ENABLE" (Bật).

Bước 4: Tạo mã API Key:
- Ở menu bên trái, chọn mục "Credentials" (Thông tin xác thực).
- Nhấn vào nút "+ CREATE CREDENTIALS" ở trên cùng và chọn "API key".

Bước 5: Copy và Sử dụng:
- Một ô cửa sổ hiện ra chứa mã API (bắt đầu bằng chữ AIza...). Hãy copy mã này và dán vào ô nhập ở trên để bắt đầu sử dụng.

Lưu ý quan trọng: Google cung cấp hạn mức miễn phí rất lớn (10.000 đơn vị/ngày). Nếu bạn dùng hết, hãy tạo thêm Key mới hoặc chờ sau 24h để hệ thống tự động reset.
