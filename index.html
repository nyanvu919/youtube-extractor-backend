Đang lấy thông tin từ YouTube...

Cách lấy YouTube API Key miễn phí (Chi tiết):

Bước 1: Truy cập Google Cloud Console và đăng nhập Gmail.

Bước 2: Tạo dự án mới:
- Nhấn chọn dự án ở góc trên cùng bên trái.
- Chọn "New Project", đặt tên bất kỳ và nhấn "Create".

Bước 3: Bật YouTube API:
- Tìm kiếm "YouTube Data API v3" trên thanh tìm kiếm.
- Chọn kết quả và nhấn nút xanh "ENABLE".

Bước 4: Tạo mã Key:
- Vào mục "Credentials" ở menu bên trái.
- Nhấn "+ CREATE CREDENTIALS" -> chọn "API key".

Bước 5: Hoàn tất:
- Copy mã AIza... hiện ra và dán vào ô nhập phía trên.

Lưu ý: Google cung cấp 10.000 đơn vị miễn phí mỗi ngày. Nếu hết, hãy tạo Key mới hoặc chờ 24h để reset.
